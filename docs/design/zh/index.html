engram Web UI

DESIGN §7.1 定义的 11 个页面的静态 HTML 原型。真实实现将使用 FastAPI + SvelteKit，通过 SSE 提供实时数据。

11 个页面

点击任意卡片查看原型
总览面板
项目级概览 — 资产计数、智慧指标折线图、活动记录、待处理提案。
/dashboard
知识图谱
可视化资产图 — D3 力导向节点，按子类型着色，含筛选侧边栏和详情面板。
/graph
记忆详情
frontmatter 编辑器、正文编辑器、入向引用、置信分仪表盘和使用历史。
/memory/<id>
工作流详情
分标签页展示 workflow.md、spine、fixtures、指标及版本时间线，含 diff 查看器。
/workflow/<name>
知识文章
三栏阅读器 — 目录侧边栏、文章正文和元数据面板（含编译状态）。
/kb/<topic>
池管理
订阅池列表、传播模式控制、依赖关系图及同步操作。
/pools
项目总览
本机所有启用 engram 的项目网格 — 资产计数、健康状态、智慧指标折线图和项目切换器。
/projects
关键页面
上下文预览
模拟 LLM 实际看到的内容 — 任务输入、token 预算、候选排名、打包后的提示词。这是最重要的诊断页面。
/context-preview
自学习控制台
实时 evolution.tsv 日志流、指标趋势图，以及自学习引擎当前轮次的 diff 查看器。
/autolearn
收件箱
跨仓消息中心 — 筛选、阅读、确认、标记已解决或拒绝跨仓传信器的消息。
/inbox
智慧指标
四组智慧指标曲线 — 工作流掌握度、任务效率、记忆质量比、上下文效率。
/wisdom
这些是静态原型

每个页面展示的都是示例数据。真实应用（里程碑 M7）将通过 FastAPI 接口和 SvelteKit 组件从 graph.db 读取实时数据。原型中的交互均不可用 — 按钮、标签页和链接仅在原型集内部跳转。
