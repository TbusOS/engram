engram Web UI

Static HTML previews of the 11 pages defined in DESIGN §7.1. The real implementation will be FastAPI + SvelteKit with live data over SSE.

These are static mockups

Every page here shows representative data. The real application (milestone M7) will render live data from graph.db via FastAPI endpoints and SvelteKit components. Interactions are non-functional in these mockups — buttons, tabs, and links navigate within the mockup set only.
